# Salary Prediction using Linear Regression

## Project Overview

This machine learning project predicts salary based on years of experience using Simple Linear Regression.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Dataset

Salary_Data.csv containing YearsExperience and Salary columns.

## Steps Performed

1. Data Loading
2. Data Visualization
3. Train-Test Split
4. Model Training
5. Prediction
6. Evaluation

## Output

The model predicts employee salary based on experience.
